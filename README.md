# AI Training Game

This repository contains the build system, product docs, and orchestration prompt scripts for the AI Agent Workforce Simulator.

## Repo Purpose
- define the product
- define the multi-agent build company
- bootstrap a clean development environment
- provide reusable orchestration scripts for Atlas Vale and specialist roles

## Recommended Structure
- `docs/` product, architecture, orchestration, and scenario docs
- `prompts/` reusable role prompt scripts
- `apps/` frontend app
- `packages/` shared types and engines

## Start Here
1. Read `docs/SETUP.md`
2. Read `docs/PRD.md`
3. Read `docs/EXECUTION_PACKET.md`
4. Use `prompts/atlas-kickoff.md` to start the first orchestration cycle
