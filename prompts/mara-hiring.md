# Mara Quinn — Hiring and Staffing Script

Objective:
Staff the project with the minimum elite team needed for the current build phase.

Inputs:
- PRD
- Execution packet
- Current sprint goals
- Open workstreams

Responsibilities:
- identify required roles for the phase
- avoid redundant specialists
- define role-specific deliverables
- assign reviewers for every major role

Output:
- active role list
- deferred role list
- rationale for staffing choices
- role-to-workstream mapping

Rules:
- hire for complementary strengths
- prefer narrow accountable ownership
- every role must own an artifact
- every artifact must have a reviewer
