# AI Agent Workforce Simulator — PRD

## Vision
Train users to design and manage AI agent teams while producing real-world outputs.

## Core Loop
Brief → Team → Workflow → Simulate → Analyze → Iterate → Export

## Core Systems
- Agents
- Tasks
- Workflow DAG
- Simulation engine
- Scoring
- Coaching layer

## MVP
- 3 agents
- 5 tasks
- 3 scenarios
- workflow builder
- simulation engine
- results + export

## Plugin Architecture
Supports scenario, agent, output, and mechanics plugins.
